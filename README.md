# OOP in pokemon world

Based on video
https://www.youtube.com/watch?v=CyRtTwKeulE
implementation of proposed architecture

## Build and run
```bash
# Clone the repository
git clone https://github.com/tomekwes/pokemon.git shelly
cd pokemon

# Configure and Build
cmake -S . -B build
cmake --build build

# Run the shell
./build/bin/pokemon
```
